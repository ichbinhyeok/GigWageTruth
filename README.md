# GigWageTruth

Discover your TRUE hourly wage as a gig worker. Don't let gross revenue fool you.

## Live Demo
This repository contains the backend and calculation logic for a gig worker profit calculator. Live site: https://gigverdict.com

The tool estimates real hourly earnings after gas, vehicle costs, and taxes for platforms like Uber and DoorDash.
